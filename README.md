## Building from Source

</div>

- Fetch latest source code from main branch.

```
git clone https://github.com/Khoa-Dam/notes_dynamic
cd notes_dynamic
```

- Rename **.env.example** => **.env**, add your own environment variables.

- Run the app with VS Code or the command line:

```
bun i || pnpm i || npm i || yarn
bun dev || pnpm dev || npm run dev || yarn dev
```
